## Up and running
nella root del progetto lanciare i seguenti comandi
```sh
# install dependecies
yarn 

# link packages 
yarn bootstrap

# Build fragments
yarn build

# Start the app
yarn start
```
Visita `http://localhost:3000` per accedere all'app.
